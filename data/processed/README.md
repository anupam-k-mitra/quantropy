# data/processed/

Processed parquet files generated by the pipeline scripts. All files here are **derived outputs** — regenerate them by running the pipelines.

| File | Generated by | Contents |
|---|---|---|
| `prices_us.parquet` | `src/data/preprocess.py` | Combined adjusted close prices — US universe |
| `prices_india.parquet` | `src/data/preprocess.py` | Combined adjusted close prices — India universe |
| `features.parquet` | `src/data/preprocess.py` | 38-signal feature matrix (5 families) |
| `regimes.parquet` | `pipelines/rq2_*/` | HMM/GMM regime labels (US + India) |

> All `.parquet` and `.csv` files in this folder are excluded from version control via `.gitignore`.
